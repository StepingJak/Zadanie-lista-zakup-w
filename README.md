Strona prezentująca manipulację DOM za pomocą jQuery.
Zawiera ona:
  -Przyciski do dodawania, usuwania, czyszczenia i przywracania produktów z/do listy.
  -Opcja filtrowania, kolorowania produktów
  -Opcje przeciągania produktów po liście za pomocą biblioteki jQuery UI (Sortable) (https://jqueryui.com/sortable/)
  -Opcja edycji produktu poprzez wciśnięcie go.

Użyte metody jQuery to: append(), prepend(), remove(), empty(), css(), sortable() (Sortable).
